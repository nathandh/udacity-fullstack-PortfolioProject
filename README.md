# udacity-fullstack-PortfolioProject
Portfolio Project for the Udacity's FullStack NanoDegree Program

This project makes use of Node Package Manager (npm) for installing and managing project dependencies. As such, please note that you will require "npm" to be installed on your system to download and populate required libraries such as: Bootstrap, Grunt, jQuery, etc.

Steps to deploy this project:

1) "git clone" this repository
2) Navigate to the top level directory and run the command "npm install"
